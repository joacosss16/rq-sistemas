# Sistema RQ — Grupo Copacabana

Digitalización del proceso de requerimientos de materiales. El contexto completo del proyecto está en **CLAUDE.md** (Claude Code lo lee automáticamente al iniciar).

## Uso rápido del prototipo
1. Abrir `prototipo/sistema_rq.html` con doble clic (Chrome/Edge, funciona offline).
2. Ingresar: usuario `gerencia`, contraseña `1234` (ver los 6 usuarios demo en la pantalla de login).
3. Los datos se guardan en el navegador automáticamente.

## Trabajar con Claude Code
```cmd
cd C:\Proyectos\sistema-rq
claude
```
Primer mensaje sugerido: "Lee CLAUDE.md y confirma que entiendes el sistema antes de empezar."
